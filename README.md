# TPTSVM
半监督学习和迁移学习相结合的跨领域及其学习方法

## Running the models
程序会在D盘下新建文件夹PTSVMtemp<br>
Windows下运行main.cpp,需要vc6.0以上版本。
###参数
1. argv1：源领域
2. argv2：目标领域标记数据
3. argv3：目标领域未标记数据
4. argv4: 中间结果输出路径
> 控制台输出文件<br>
